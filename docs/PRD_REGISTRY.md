# PRD 总集

| 版本 | 标题 | 需求内容（详细摘要） | PRD链接 |
| --- | --- | --- | --- |
| PRD-STORY-VARIANTS-V1 | OldCheat 可持续故事变体训练 - V1 | 为现有 14 个诈骗大类增加可维护的受控故事变体，在每次训练开始时优先抽取近期未体验变体，并锁定人物、背景、目标、压力手法、开场白和 fallback。模型与 RAG 负责在剧本卡边界内自然演绎，模型、RAG 或在线存储异常时自动回退到同变体 fallback、内置种子或原静态场景。范围包括文字与语音入口统一抽取、报告记录、防立即重复、轻量内部维护页面、安全校验、功能开关和兼容回滚；不包含无限自由生成、真实犯罪信息、复杂 CMS 或完整用户账户系统。 | docs/prd/PRD-STORY-VARIANTS-V1.md |
| PRD-VOICE-CALL-V3 | 银龄反诈实时语音通话训练模式 - V3 | 在现有 OldCheat 训练台和 V1 语音训练基础上，升级为更接近真实电话的实时语音训练模式。核心范围包括电话式主界面、云端实时 ASR、流式 TTS、通话状态机、求助和挂断安全按钮、训练报告语音记录、成本控制和云服务失败降级。边界是不接真实电话线路、不做 WebRTC 多人通话、不长期保存原始音频、不把阿里云密钥暴露给浏览器。实现时必须保留现有文字训练、场景库、DeepSeek/RAG、风险识别和报告能力。 | docs/prd/PRD-VOICE-CALL-V3.md |
| PRD-MOBILE-ELDERLY-VOICE-UI | 银龄反诈手机端老年人极简语音训练 UI - V1 | 在同一个 Netlify 地址下为手机用户新增老年人极简语音训练流程，电脑端继续保留现有完整训练台。核心范围包括手机端完整场景选择、类别分组与高风险优先、训练中换场景二次确认、场景准备页、大号开始语音训练和重新开启麦克风按钮、电话式语音训练页、再说一遍、向子女求助、挂断退出、训练结束页、手机适配报告和响应式分流。实现时必须共用现有动态场景库、DeepSeek/RAG、风险识别和报告链路，不新增第二个网址，不破坏电脑端三栏训练台，不暴露任何 API key。 | docs/prd/PRD-MOBILE-ELDERLY-VOICE-UI.md |
